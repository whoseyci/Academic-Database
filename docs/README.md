# Academic Database GitHub Pages

This folder is served by GitHub Pages.

- `index.html` — static claim-ledger interface.
- `data/harness_export.json` — exported database contents for the UI.
- `CLAIM_GUIDELINES.md` — straightforward claim classification guidelines.

The static export intentionally does **not** include full source papers. It includes metadata, claims, evidence excerpts, tags, and bounded context snippets.
